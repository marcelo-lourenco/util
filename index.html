Ferramentas Úteis para Testes e Desenvolvimento

Uma coleção de utilitários online, extensões e pacotes para agilizar tarefas comuns em testes de software e desenvolvimento, com foco em dados brasileiros e CNAB.

Apoio aos Testes

Gerador de Dados Brasileiros

CPF, CNPJ, RG, CNH, PIS, Contas, CEP e mais.

CNAB Layout Colorido

Visualizador de arquivos CNAB (150, 240, 400) com destaque de campos.

CNAB 240: Remessa p/ Retorno

Converte arquivos de remessa (.REM) em arquivos de retorno (.RET).

CNAB 240: Localizar NUMDOC

Encontra NUMDOC em arquivos com Segmentos A e J.

CNAB 240: Selecionar Colunas

Extrai colunas específicas de arquivos com Segmentos A e J, O e N

MongoDB: Gerar Docs

Cria documentos para a collection "controle_lancamentos".

Validador CNPJ Alfanumérico

Valida CNPJs que contêm letras e números.

Extensões Chrome

CNAB: Remessa p/ Retorno

Extensão para converter arquivos CNAB .REM em .RET.

Validador de Boleto

Converte e valida códigos de barras e linhas digitáveis.

Gerador de Dados Fictícios

Gera diversos tipos de dados brasileiros diretamente no navegador.

Jira: Expandir Área

Otimiza a visualização de issues no Jira expandindo a área útil.

Pacotes e Extensões Dev

NPM: Gerador BR

Pacote Node.js para gerar dados brasileiros em testes automatizados.

VS Code: API Test Builder

Converte Swagger/OpenAPI em scripts de teste (Playwright/Cypress).

VS Code: Live API & URL Monitor

Live monitor for your API endpoints and URLs. Check status, health, and availability in real-time, all within VS Code.
